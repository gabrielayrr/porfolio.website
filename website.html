02.01.02 WEBSITE

TEMA 02

II tema 02 blev vi introduceret til grundlaget for multimediedesign og de mest almindeligt anvendte værktøjer, der bruges som multimediedesigner og til opbygning af hjemmesider, fx html, css og grid. Hvordan man gør den responsiv samt idéprocessen og styling.

GÅ TIL HJEMMESIDEN

STUDIESTARTSPROVEN

I dette projekt lavede vi vores første responsive hjemmeside. Vi fik alt materialet fra vores lærere og skulle følge et layoutdiagram og wireframes og placere tekst og billeder på det rigtige sted ved hjælp af grid og flexbox i css. Vi havde frie hænder til at style hjemmesiden og skulle gøre den mobilvenlig først.
